# Retail Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data to identify sales performance, profitability, customer behavior, and shipping trends using Power BI.

## Business Problem

Business managers need a centralized dashboard to monitor:

- Total Sales
- Profit
- Order Volume
- Customer Performance
- Product Profitability
- Shipping Performance

## Dataset

Superstore Retail Dataset

## Tools Used

- Power BI
- Excel
- Data Modeling
- DAX
- Data Visualization

## Dashboard Features

- KPI Cards
- Interactive Slicers
- Sales Trend Analysis
- Segment Analysis
- Shipping Analysis
- Customer Analysis
- Product Profitability

## Key Insights

- Consumer segment contributes the highest sales.
- Standard Class is the most used shipping mode.
- A small number of customers contribute significantly to revenue.
- Copiers generate the highest profit among sub-categories.
- Sales increase steadily across quarters.
